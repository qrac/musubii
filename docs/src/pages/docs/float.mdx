# Float

contents list

## Basic

- `.is-float-right` で `float: right;` を設定します
- `.is-float-left` で `float: left;` を設定します
- `.is-float-none` で `float: none;` を設定します
- `.is-clearfix` を付与した要素の後で Float を解除します
