# Joint

直下の [Button](/docs/button)・[Badge](/docs/badge)・[Form](/docs/form) の `.input` や `.select` を連結させられるレイアウト CSS。

## Table of Contents

contents list

## Basic

- `.joint` を設置し直下に `.button` `.badge` `.input` `.select` を配置します
- 自動的に角丸と区切りのボーダーが調整されます

{/* prettier-ignore */}
<div className="docs-preview">
  <div className="joint">
    <button className="button is-outline" type="button">ボタン</button>
    <button className="button is-outline" type="button">ボタン</button>
    <button className="button is-outline" type="button">ボタン</button>
  </div>
</div>

## Import

PostCSS で読み込む場合の各パスは以下となります。

```css

```
