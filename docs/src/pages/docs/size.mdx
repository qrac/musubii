# Size

エレメントのサイズを変更するユーティリティ CSS。

## Table of Contents

contents list

## Basic

- `.is-{size}` で `font-size` を変更します
- `{size}` は最大 `xxl` から最小 `xxs` まで
- エレメントのほとんどは em で設計されているため相対的にサイズ変更できます
- 各クラスは画面サイズに応じて適応できます

| サイズ               | すべて    | 576px 以上       | 768px 以上       | 992px 以上        | 1200px 以上    |
| -------------------- | --------- | ---------------- | ---------------- | ----------------- | -------------- |
| <small>200%</small>  | `.is-xxl` | `.fablet:is-xxl` | `.tablet:is-xxl` | `.desktop:is-xxl` | `.wide:is-xxl` |
| <small>150%</small>  | `.is-xl`  | `.fablet:is-xl`  | `.tablet:is-xl`  | `.desktop:is-xl`  | `.wide:is-xl`  |
| <small>125%</small>  | `.is-lg`  | `.fablet:is-lg`  | `.tablet:is-lg`  | `.desktop:is-lg`  | `.wide:is-lg`  |
| <small>100%</small>  | `.is-md`  | `.fablet:is-md`  | `.tablet:is-md`  | `.desktop:is-md`  | `.wide:is-md`  |
| <small>87.5%</small> | `.is-sm`  | `.fablet:is-sm`  | `.tablet:is-sm`  | `.desktop:is-sm`  | `.wide:is-sm`  |
| <small>75%</small>   | `.is-xs`  | `.fablet:is-xs`  | `.tablet:is-xs`  | `.desktop:is-xs`  | `.wide:is-xs`  |

## Import

PostCSS で読み込む場合の各パスは以下となります。

```css
@import "musubii/src/utilities/size/main.css";
@import "musubii/src/utilities/size/fablet.css";
@import "musubii/src/utilities/size/tablet.css";
@import "musubii/src/utilities/size/desktop.css";
@import "musubii/src/utilities/size/wide.css";
```
