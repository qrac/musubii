# Button

ボタンを装飾するエレメント CSS。

## Table of Contents

contents list

## Basic

- `.button` がベースとなりサイズなどは設定されますが単体では完成しません
- `.is-plain` または `.is-outline` または `.is-melt` を併用してください

{/* prettier-ignore */}
<div className="docs-preview">
  <button className="button" type="button">ボタン</button>
</div>

{/* prettier-ignore */}
```html
<button class="button" type="button">ボタン</button>
```

## Plain

- `.is-plain` でプレーンタイプのボタン装飾を作ります
- `.is-{color}` でボタンに色を付けます
- 色は `primary` `secondary` `info` `success` `warning` `danger` の 6 種類

{/* prettier-ignore */}
<div className="docs-preview">
  <div className="box is-flex is-gap-xs">
    <button className="button is-plain" type="button">詳細</button>
    <button className="button is-plain is-primary" type="button">決定</button>
    <button className="button is-plain is-secondary" type="button">変更</button>
    <button className="button is-plain is-info" type="button">情報</button>
    <button className="button is-plain is-success" type="button">送信</button>
    <button className="button is-plain is-warning" type="button">注意</button>
    <button className="button is-plain is-danger" type="button">削除</button>
  </div>
</div>

{/* prettier-ignore */}
```html
<button class="button is-plain" type="button">詳細</button>
<button class="button is-plain is-primary" type="button">決定</button>
<button class="button is-plain is-secondary" type="button">変更</button>
```

## Import

PostCSS で読み込む場合の各パスは以下となります。

```css
@import "musubii/src/elements/button/main.css";
```

v7 を再現する場合は `legacy.css` を追加で読み込んでください。

```css
@import "musubii/src/layouts/button/legacy.css";
```
