# Button

ボタンの装飾を作るエレメント CSS。

## Table of Contents

contents list

## Basic

- `.button` がベースとなります
- `.button` によってサイズなどは設定されますが単体では完成しません
- 次項の Variants を併用してください

{/* prettier-ignore */}
<div className="docs-preview">
  <button className="button" type="button">ボタン</button>
</div>

{/* prettier-ignore */}
```html
<button class="button" type="button">ボタン</button>
```

## Variants

- `.is-plain` でプレーンタイプのボタン装飾を作ります
- `.is-outline` でアウトラインタイプのボタン装飾を作ります
- `.is-melt` で溶け込むタイプのボタン装飾を作ります
- `.is-{color}` を追加するとボタンに色が付きます
- `{color}` は `primary` `secondary` `info` `success` `warning` `danger` の 6 種類

## Angle

- `.is-angle-right` で右に角アイコンを表示させます
- `.is-angle-left` で左に角アイコンを表示させます
- `.is-angle-up` で角アイコンを上向きにします
- `.is-angle-bottom` でで角アイコンを下向きにします
- 角アイコン分の余白が足りない場合は [padding](/docs/padding) で設定します

## Arrange

- ボタンの子要素の配置を調整します

| クラス名       | 効果                         |
| -------------- | ---------------------------- |
| `.is-baseline` | 子要素をベースラインに揃える |
| `.is-left`     | 子要素を左に揃える           |
| `.is-right`    | 子要素を右に揃える           |

{/* prettier-ignore */}
<div className="docs-preview">
  <button className="button is-plain is-primary is-gap-none is-baseline" type="button">
    <span className="text is-lg">登録する</span>
    <span className="text is-xs">（無料）</span>
  </button>
  <div className="box is-flex is-gap-xs is-mt-sm">
    <button className="button is-plain is-primary is-flex-0 is-left" type="button">
      <svg className="icon" viewBox="0 0 100 100" xmlns="http://www.w3.org/2000/svg">
        <path d="M 10 30 A 20 20 0 0 1 50 30 A 20 20 0 0 1 90 30 Q 90 60 50 90 Q 10 60 10 30 Z" />
      </svg>
      <span className="text">ボタン</span>
    </button>
    <button className="button is-plain is-primary is-flex-0 is-right" type="button">
      <span className="text">ボタン</span>
      <svg className="icon" viewBox="0 0 100 100" xmlns="http://www.w3.org/2000/svg">
        <path d="M 10 30 A 20 20 0 0 1 50 30 A 20 20 0 0 1 90 30 Q 90 60 50 90 Q 10 60 10 30 Z" />
      </svg>
    </button>
  </div>
</div>

{/* prettier-ignore */}
```html
<button class="button is-plain is-primary is-baseline" type="button">...</button>
<button class="button is-plain is-primary is-left" type="button">...</button>

```

## Disabled

- `[disabled]` または `[aria-disabled="true"]` または `.is-disabled` で非活性化します
- 非活性化すると `pointer-events: none;` が付与されホバー効果などが打ち消されます
- 非活性化した `.button` は専用の色味に変化します

{/* prettier-ignore */}
<div className="docs-preview">
  <div className="box is-flex is-gap-xs">
    <button className="button is-plain" type="button" disabled>詳細</button>
    <button className="button is-plain is-primary" type="button" disabled>決定</button>
    <button className="button is-outline" type="button" disabled>詳細</button>
    <button className="button is-outline is-secondary" type="button" disabled>変更</button>
    <button className="button is-melt" type="button" disabled>詳細</button>
