# Setup

## CDN

以下リンクで基本構成（ドキュメント記載のすべての CSS）が使えます。ダークテーマは `data-theme="dark"` の付与で反映されます。

{/* prettier-ignore */}
```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/musubii@8.0.0/dist/musubii.css" />
```

## npm

基本構成をすべてを利用する場合は PostCSS の処理の中で `musubii` をインポート。

```sh
$ npm install musubii
```

```css
@import "musubii";
```

部分的に利用する場合は以下から必要なものを選択してください。
