# Setup

MUSUBii は CDN または npm で利用できます。

## Table of Contents

contents list

## CDN

各バージョンを <ExternalLink href="https://www.jsdelivr.com/package/npm/musubii">jsDelivr</ExternalLink> のリンクで使えます。

{/* prettier-ignore */}
```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/musubii@9.2.0/dist/musubii.css" />
```

## npm

各バージョンを <ExternalLink href="https://www.npmjs.com/package/musubii">npm</ExternalLink> からインストールできます。

```sh
$ npm install musubii
```

すべての CSS を利用する場合は PostCSS の処理の中で `musubii` をインポート。

```css
@import "musubii";
```
