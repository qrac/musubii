# Migration

ここでは v8 から v9 への移行方法を説明します。v7, v7-alpha.41~42 から v8 への移行方法は[アーカイブドキュメント](https://musubii-archive-v8-6-0.netlify.app/docs/migration)を参照してください。

## Table of Contents

contents list

## CDN

v9 で使えなくなったクラス名はありません。CDN を利用されている場合は読み込んでいるパスを変更するだけで移行完了です。

`migrate.css` を追加で読み込むことで v7 の色や サイズ感を再現できます。

{/* prettier-ignore */}
```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/musubii@9.2.0/dist/musubii.css" />
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/musubii@9.2.0/dist/migrate.css" />
```

## npm

### Upgrade MUSUBii

MUSUBii を v9 にアップグレード。

```sh
$ npm install musubii@latest
```
