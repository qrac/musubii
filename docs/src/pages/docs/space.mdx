# Space

直下要素の間にマージンを指定するユーティリティ CSS。

## Table of Contents

contents list

## Basic

- `.is-space-{size}` で直下の最初の要素以外に `margin-top` を付与します
- `{size}` は最大 `xxl` から最小 `xxs` まで

| サイズ              | クラス名        |
| ------------------- | --------------- |
| <small>28px</small> | `.is-space-xxl` |
| <small>24px</small> | `.is-space-xl`  |
| <small>20px</small> | `.is-space-lg`  |
| <small>16px</small> | `.is-space-md`  |
| <small>12px</small> | `.is-space-sm`  |
| <small>8px</small>  | `.is-space-xs`  |
| <small>4px</small>  | `.is-space-xxs` |

## Way

- `.is-space-{way}-{size}` で直下のすべての要素に `margin-{way}` を付与します
- `{way}` は `top` `right` `bottom` `left` の 4 種類
- `{size}` は最大 `xxl` から最小 `xxs` まで

| サイズ              | Top                 | Right                 | Bottom                 | Left                 |
| ------------------- | ------------------- | --------------------- | ---------------------- | -------------------- |
| <small>28px</small> | `.is-space-top-xxl` | `.is-space-right-xxl` | `.is-space-bottom-xxl` | `.is-space-left-xxl` |
| <small>24px</small> | `.is-space-top-xl`  | `.is-space-right-xl`  | `.is-space-bottom-xl`  | `.is-space-left-xl`  |
| <small>20px</small> | `.is-space-top-lg`  | `.is-space-right-lg`  | `.is-space-bottom-lg`  | `.is-space-left-lg`  |
| <small>16px</small> | `.is-space-top-md`  | `.is-space-right-md`  | `.is-space-bottom-md`  | `.is-space-left-md`  |
| <small>12px</small> | `.is-space-top-sm`  | `.is-space-right-sm`  | `.is-space-bottom-sm`  | `.is-space-left-sm`  |
| <small>8px</small>  | `.is-space-top-xs`  | `.is-space-right-xs`  | `.is-space-bottom-xs`  | `.is-space-left-xs`  |
| <small>4px</small>  | `.is-space-top-xxs` | `.is-space-right-xxs` | `.is-space-bottom-xxs` | `.is-space-left-xxs` |
