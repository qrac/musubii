# Introduction

MUSUBii（むすびー）は <ExternalLink href="https://yakuhanjp.qranoko.jp/">YakuHanJP</ExternalLink> を開発しているデザイナーの <ExternalLink href="https://twitter.com/Qrac_JP">クラク</ExternalLink> が作った CSS フレームワークです。マークアップを覚え始めた 2016 年初頭から自分用に開発を始めました。

デザインや機能面のコーディングがフレームワークに依存しないよう、大きなコンポーネント・JavaScript は含まずベース CSS + α というミニマムな構成にしています。

MUSUBii を使うだけでサイトは完成しませんが、毎度同じような CSS を書いている時間を大幅に減らすことができ、疲弊することなくデザイン・コーディングを進められます。

日本のウェブデザイナーが腹ごなしに MUSUBii を使い、元気にデザインしてくれたら幸いです。

## Layers

CSS は大きく 4 つに分類されます。

| Name        | Detail                                        |
| ----------- | --------------------------------------------- |
| `bases`     | 下地となる色変数・文字変数・リセット CSS など |
| `layouts`   | セクション・グリッドシステムなど              |
| `elements`  | テキスト・ボタン・フォームなど                |
| `utilities` | 調整用のモディファイア                        |

## Media Query

`utilities` の一部はメディアクエリによって 5 つの画面サイズで有効化できます。

| Name      | Width             |
| --------- | ----------------- |
| `mobile`  | `0 <= width`      |
| `fablet`  | `576px <= width`  |
| `tablet`  | `768px <= width`  |
| `desktop` | `992px <= width`  |
| `wide`    | `1200px <= width` |
