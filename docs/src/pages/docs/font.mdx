# Font

下地となるフォントの変数。上書きすることで全体のフォント設定をカスタマイズできます。

## Table of Contents

contents list

## Variables

上書きできる変数は以下の通りです。

```css
:root {
  --font-sans: "Hiragino Sans", "Noto Sans JP", Meiryo, sans-serif;
  --font-sans-en:
    system-ui, "Segoe UI", Roboto, "Helvetica Neue", Arial, "Noto Sans",
    sans-serif;
  --font-serif:
    "Hiragino Mincho ProN", "Noto Serif JP", "Yu Mincho", YuMincho, serif;
  --font-serif-en: Georgia, Cambria, "Times New Roman", Times, serif;
  --font-mono:
    SFMono-Regular, Menlo, Monaco, Consolas, "Liberation Mono", "Courier New",
    monospace;
}
```

## Import

PostCSS で読み込む場合のパスは以下となります。

```css
@import "musubii/src/bases/typo/font.css";
```
