# Font

下地となるフォントの変数。上書きすることで全体のフォント設定をカスタマイズできます。

## Table of Contents

contents list

## Variables

上書きできる変数は以下の通りです。

```css
:root {
  --font-sans: "Hiragino Sans", "Hiragino Kaku Gothic ProN",
    "Noto Sans Japanese", Meiryo, "Yu Gothic Medium", sans-serif,
    "Apple Color Emoji", "Segoe UI Emoji", "Segoe UI Symbol", "Noto Color Emoji";
  --font-sans-en: system-ui, -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto,
    "Helvetica Neue", Arial, "Noto Sans", sans-serif, "Apple Color Emoji", "Segoe UI Emoji",
    "Segoe UI Symbol", "Noto Color Emoji";
  --font-serif: "Hiragino Mincho ProN", "Noto Serif JP", "Yu Mincho", YuMincho,
    serif;
  --font-serif-en: Georgia, Cambria, "Times New Roman", Times, serif;
  --font-mono: SFMono-Regular, Menlo, Monaco, Consolas, "Liberation Mono",
    "Courier New", monospace;
}
```

## Import

PostCSS で読み込む場合の各パスは以下となります。

```css
@import "musubii/src/bases/typo/font.css";
```
