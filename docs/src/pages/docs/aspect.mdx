# Aspect

アスペクト比を設定するユーティリティ CSS。

## Table of Contents

contents list

## Basic

- `.is-aspect-{ratio}` で縦横比を固定できます
- `{ratio}` は `16x9` `3x2` `4x3` `1x1` の 4 種類

| アスペクト比        | クラス名          |
| ------------------- | ----------------- |
| <small>16x9</small> | `.is-aspect-16x9` |
| <small>3x2</small>  | `.is-aspect-3x2`  |
| <small>4x3</small>  | `.is-aspect-4x3`  |
| <small>1x1</small>  | `.is-aspect-1x1`  |
